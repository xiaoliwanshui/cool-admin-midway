# Cool Admin Skills

## Available Skills

### Core Skills
- **authority**: Authority and permission management using JWT
- **module**: Module development and configuration
- **controller**: API controller development
- **service**: Business logic service layer
- **db**: Database operations and entity management

### Advanced Skills
- **task**: Task scheduling and queue processing
- **cache**: Caching strategies and optimization
- **event**: Event-driven programming
- **exception**: Exception handling and error management
- **socket**: WebSocket real-time communication
- **tenant**: Multi-tenant architecture

## Usage
Invoke skills when you need to:
- Implement authentication and authorization
- Create new modules or configure existing ones
- Develop API endpoints and controllers
- Implement business logic in services
- Work with databases and entities
- Set up scheduled tasks or queues
- Optimize performance with caching
- Implement event-based architectures
- Handle exceptions and errors
- Add real-time communication
- Build multi-tenant applications

## Skill Structure
Each skill includes:
- Overview of the functionality
- Configuration and setup instructions
- Code examples
- Best practices
- Common use cases

## Getting Started
1. Choose the relevant skill based on your task
2. Follow the setup instructions
3. Use the code examples as templates
4. Apply best practices for optimal results
